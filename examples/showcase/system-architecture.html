Acme API — System Architecture

High-level service map. Edge → Gateway → Services → Data. Internal traffic stays in the VPC; public edge handled at Cloudflare.

Components

API gateway Rust · Axum

Single ingress for service traffic. Routing, auth header validation, rate limits, request tracing.

auth-svc Rust

Magic-link issue/verify, WebAuthn ceremony, session lookup. Reads/writes sessions + users.

billing-svc Go

Stripe webhooks, plan transitions, usage metering. Idempotent on event id.

webhook-svc Go

Outbound webhooks to customer endpoints, exponential retries up to 24h, dead-letter to Kafka.

Postgres RDS · 16

Primary OLTP store. Logical replication to analytics warehouse hourly.

Redis ElastiCache

Session cache, rate-limit counters, idempotency keys (24h TTL).
